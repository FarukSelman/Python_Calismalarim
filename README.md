# 🐍 Python Core Algorithms & Study Notes

Bu depo, Python programlama dilini öğrenme sürecinde geliştirdiğim temel algoritmaları, problem çözme çalışmaları ve programlama pratiklerini içermektedir. Amaç; temel Python kavramlarını pekiştirmek, algoritmik düşünme becerisini geliştirmek ve yazılım geliştirme sürecinde kullanılan temel kontrol yapıları ile çalışmaktır.

---

# 🛠️ İçerik

Depoda bulunan Python dosyaları temel algoritma problemlerinin çözümlerini içermektedir.

## 🔢 Asal Sayı Kontrolü

**`asal_sayi.py`**

Girilen bir sayının asal olup olmadığını döngüler ve koşul ifadeleri kullanarak kontrol eder.

---

## 📐 Faktöriyel Hesaplama

**`faktoriyel.py`**

Bir sayının faktöriyelini iteratif (döngü tabanlı) yöntemle hesaplar.

---

## 🔄 Palindrom Kontrolü

**`palindrom.py`**

Girilen metin veya sayının tersten okunduğunda aynı olup olmadığını kontrol eder.

---

## ⚖️ Tek / Çift Sayı Kontrolü

**`tek_cift.py`**

Mod alma (`%`) operatörünü kullanarak girilen sayının tek mi yoksa çift mi olduğunu belirler.

