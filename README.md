# 🐍 Python Core Algorithms & Study Notes

Bu depo; yazılım mühendisliği eğitimim sürecinde temel programlama pratiklerini, matematiksel mantık yürütme süreçlerini ve veri manipülasyon tekniklerini geliştirmek amacıyla saf (**Vanilla**) Python kullanarak geliştirdiğim temel algoritma çözümlerini ve çalışma notlarımı barındırmaktadır.

---

## 📂 Depo İçeriği ve Çözülen Problemler

Depo içerisinde yer alan scriptler ve odaklandıkları mantıksal yapılar şu şekildedir:

- 🔢 **`asal_sayi.py`:** Girilen bir sayının tam bölenlerini kontrol ederek asallık durumunu verimli döngülerle analiz eder.
- 📐 **`faktoriyel.py`:** Matematiksel faktöriyel işlemlerini döngüsel (iterative) mantık yapıları kullanarak hesaplar.
- 🔄 **`palindrom.py`:** Metinsel veya sayısal verilerin düzden ve tersten okunuş simetrisini string indeksleme teknikleriyle doğrular.
- ⚖️ **`tek_cift.py`:** Temel mod alma (modulus) operatörlerini kullanarak sayısal verilerin yapısal sınıflandırmasını yapar.

2. Bilgisayarda Çalıştırma
Projelerin çalışması için bilgisayarınızda Python 3'ün kurulu olması yeterlidir.

Projeyi bilgisayarınıza klonlayın:

```bash
git clone [https://github.com/FarukSelman/Python_Calismalarim.git](https://github.com/FarukSelman/Python_Calismalarim.git)
```
Klasörün içine geçiş yapın:

```bash
cd Python_Calismalarim
```
İstediğiniz herhangi bir algoritma dosyasını terminal üzerinden doğrudan çalıştırabilirsiniz:

```bash
python asal_sayi.py
```
